Is Procraft free? +

Yes, creating a profile and getting started with all basic templates is completely free. Additional premium features will be added later.

Where is the dashboard? +

The portfolio creation app works at dashboard.procraft.uz. All settings, templates and PDF export features are available there.

Can I change the template later? +

Yes, of course. All content is saved, only the public page appearance changes. You can switch templates at any time.
