# Shared packages (`@procraft/*`)

| Package          | Responsibility                                  |
| ---------------- | ----------------------------------------------- |
| `config`         | Constants/env/theme                             |
| `api` (@procraft/api)        | Axios + interceptors                       |
| `services`       | Endpoint modules                                |
| `hooks`          | TanStack Query facades                         |
| `store`          | Redux UI slices                                 |
| `ui`             | Stateless components                            |
| `utils`          | Pure helpers                                   |
| `i18n`           | Translations bootstrap                          |

Import rules summarized in [`../docs/FRONTEND_RULES.md`](../docs/FRONTEND_RULES.md).
